Functionality and Uses for Anime Reviewer

Our Application acts as a discussion board and journal for users to share their thoughts, recommendations, and opinions on particular animes!

Forum Functions Include:

  • Add A New Anime!
  • Edit an Old Anime
  • Delete Animes
  • Add Characters to an Anime
  • Remove Characters from an Anime
  • Edit Characters from an Anime
  • Add a Review to an Anime
  • Delete a Review from an Anime
Technologies used to create this platform include: PSQL, Django, Materialize, GoogleImageSearch API, AnimeChan API, Heroku, Css, and HTML
